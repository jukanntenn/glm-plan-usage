**1. z.ai configuration:**
```
User sets ANTHROPIC_BASE_URL=https://api.z.ai/...
  → main.rs loads Config
  → GlmUsageSegment.collect calls GlmApiClient::from_env(config)
  → Platform::detect_from_url recognizes z.ai
  → build_quota_url constructs correct endpoint
  → Request sent with configured timeout
  → Retries with configured attempt count
  → Response formatted for status bar

  → Retry loop runs configured number of attempts
```
**Result:** Complete ✅ (prior drift fixed)
