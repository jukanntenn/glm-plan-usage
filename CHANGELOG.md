# Changelog

## [0.3.1] - 2026-06-16

### Added

- `--version` flag to print the current version
- GLM-5.2 recognized as a premium model for multiplier display
- Extended off-peak 1x deduction promotional rate through 2026-09-30 (previously 2026-06-30)

## [0.3.0] - 2026-05-25

### Added

- `setup` command to automatically configure Claude Code statusline settings
- Combined mode support with CCometixLine via `--with-ccline` flag

## [0.2.1] - 2026-05-20

### Fixed

- TOML config deserialization for migrated configuration values
- Color thresholds adjusted for better status bar visibility

## [0.2.0] - 2026-05-10

### Added

- Model consumption multiplier support with configurable display and peak-hour awareness

## [0.1.0] - 2026-04-30

### Added

- Weekly quota tracking for new plan users
- Display mode improvements for status bar segments

## [0.0.3] - 2025-02-14

### Added

- Token countdown timer showing time until quota reset
- Automatic unit formatting (M/K) for token counts
- Emoji icons (🪙🌐) for clearer status bar visualization

### Fixed

- API compatibility with latest GLM platform changes
- Edge cases in token count formatting

## [0.0.2] - 2025-02-07

### Added

- Multi-platform support

## [0.0.1] - 2025-01-31

### Added

- Initial release
- GLM usage tracking (tokens and MCP)
